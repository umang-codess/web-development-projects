# Web Development Projects

A repository dedicated to practicing and storing frontend web components and single-page applications

## Projects Included

*   **portfolio.html** — Responsive personal portfolio layout
*   **login.html** — User login interface with modern styling.
*   **form.html** — Structured data registration form.
*   **card.html** — Clean card component using CSS Flexbox/Grid
*   **valentine.html** — Custom themed page with micro-interactions

##  Core Technologies
*   **HTML5** (Structure & Semantics)
*   **CSS3** (Responsive Layout & Design)
*   **JavaScript** (DOM Interactivity)
